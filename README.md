# Webcelerate

A userscript that adds shortcuts for quick model switching on AI chat interfaces. Press F1-F4 to switch models instantly â€” for example, F1 selects ChatGPT 5 Instant on chatgpt.com, or Sonnet 4.5 on claude.ai. This can be configured in the script.

Made with Claude.

## Installation

1. Install a userscript manager like Tampermonkey or Violentmonkey in your browser.
2. Open the script: https://raw.githubusercontent.com/4x1om/webcelerate/main/webcelerate.user.js
3. Your userscript manager should prompt you to install. Click "Install".

The script works automatically on chatgpt.com and claude.ai. Updates are pulled from this repository.
